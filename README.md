<h1>Snake Game</h1>
<p>Volvamos a 1998 cuando jugabamos en nuestro Nokia el famoso juego de Snake, en este proyecto se puede jugar tanto con el keyboard, como con el control integrado en la interfaz</p>

<h1>Preview del proyecto<h1>
<img align="center" src="./snakepreview.gif"/>

<h1>Funciones principales</h1>
  <ul>
    <li>Controlar el juego por teclado y el control integrado.</li>
    <li>Se debe llevar un contador de los puntos ganados en la partida.</li>
    <li>Cuando el jugador pierde, notificarlo y preguntar si quiere intentarlo de nuevo.</li>
   
  </ul>
